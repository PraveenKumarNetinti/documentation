## Where to Add Your Product Code:

### For a new product plugin:

1. **Create folder**: `products/your_product_name/`
2. **Required file**: `plugin.py` that contains a class inheriting from `apps.products.base.BaseProduct`
3. **Add your models**: `products/your_product_name/models.py`
4. **Add your views**: `products/your_product_name/views.py`
5. **Add your URLs**: `products/your_product_name/urls.py`
6. **Register in settings**: Add `'products.your_product_name'` to `PRODUCT_MODULES` in `config/settings/base.py`

### Example plugin.py:

```python
# products/your_product_name/plugin.py
from apps.products.base import BaseProduct
from apps.products.registry import register_product

@register_product
class YourProduct(BaseProduct):
    slug = 'your_product'
    name = 'Your Product Name'

    def on_install(self, organization_product):
        # Installation logic
        return True

    def on_uninstall(self, organization_product):
        # Uninstall logic
        return True
```
